# How to setup?

### Steps:
- Clone the project:
  ```bash
  git clone https://github.com/Pandepic-pp/easeops-ass-frontend.git
  ```
  
- Install the necessary dependencies:
  ```bash
  npm i
  ```

- Run the project
  ```bash
  npm run dev
  ```
